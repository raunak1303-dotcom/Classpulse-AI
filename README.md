# ClassPulse AI 🚀

ClassPulse AI is an AI-powered smart attendance management system built using Python, OpenCV, SQLite, and Tkinter.

## Features
- Face Detection
- Face Recognition
- Student Registration
- Automatic Attendance Marking
- SQLite Database Integration
- Dashboard GUI
- Attendance Viewer
- Student Management

## Technologies Used
- Python
- OpenCV
- Tkinter
- SQLite
- NumPy

## Future Improvements
- Phone Detection
- Attention Scoring
- Web Application Version
- Analytics Dashboard
- YOLO Integration
