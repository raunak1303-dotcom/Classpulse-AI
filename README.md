# ClassPulse AI 🚀

ClassPulse AI is an AI-powered smart classroom monitoring system built using Python, OpenCV, YOLOv8, SQLite, and Tkinter.

The system automatically:
- Recognizes students using Face Recognition
- Marks attendance in real-time
- Detects mobile phone usage
- Monitors classroom engagement
- Calculates attention scores
- Generates attendance & engagement reports

---

## ✨ Features

- 🎯 Automatic Face Recognition Attendance
- 📱 Real-Time Phone Detection
- 🧠 Student Engagement Monitoring
- ⚠ Live Attention Score Alerts
- 📊 Attendance & Engagement Reports
- 🖥 Modern Teacher Dashboard
- 📁 Excel Report Export

---

## 🛠 Technologies Used

- Python

- YOLOv8
- SQLite
